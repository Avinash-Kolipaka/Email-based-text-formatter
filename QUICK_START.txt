→ Target: 60-85 for business emails

TIPS & TRICKS
─────────────────────────────────────────────────────────────────────────────

✅ DO:
• Use clear, concise input text
• Choose the appropriate tone for your audience
• Review analytics before sending
• Save important emails as drafts
• Use templates for recurring email types
• Check subject line before copying

❌ DON'T:
• Use ALL CAPS (system handles formatting)
• Include HTML or special formatting (keep it plain text)
• Expect 100% accuracy (always proofread)
• Rely on system for very technical terminology
• Send without reviewing the output

🎯 OPTIMIZATION:
• Aim for Readability Score 65-80
• Aim for Politeness Score 70-90
• Keep sentences under 20 words
• Keep paragraphs 3-5 sentences
• Use simple, common words
• Break complex ideas into separate sentences

TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

Problem: "Cannot reach server"
Solution: Make sure Flask is running
          python app.py in terminal

Problem: "Styles look wrong"
Solution: Hard refresh browser (Ctrl+Shift+R)

Problem: "Drafts disappeared"
Solution: Check if private/incognito mode
          (disables local storage)

Problem: "Export to Gmail not working"
Solution: Check browser pop-up blocker
          Allow pop-ups for 127.0.0.1

Problem: "Text not formatting"
Solution: Make sure text field has content
          Click format button
          Check Flask server logs
